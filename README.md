# FancyFormatter

FancyFormatter is a Sublime Text 4 plugin for formating.

support for the following languages: If you want to add other languages , please tell me.
- json
- proto
- javascript
- typescript
- html
- css
- less
- sass
- python
- go
- php
- markdown
- xml
- yaml
- cpp
- c
